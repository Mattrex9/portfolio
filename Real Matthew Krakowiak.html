Masters Tournament Bracket App

A shipped JavaScript project built around live sports data, user picks, leaderboard tracking, and custom scoring logic.

The idea

I wanted to make watching the Masters more competitive with my friends by letting users pick golfers and track how their picks perform in real time. Instead of a static bracket, the app updates around live leaderboard data and calculates points from each golfer's position.

What I built

  • A live web app where users can pick golfers and compare performance during the tournament.
  • Custom scoring logic that awards points based on leaderboard position.
  • Integration with ESPN leaderboard data to keep standings current.
  • A user interface focused on clarity: picks, rankings, scoring, and tournament movement.

Next Project

AI Guitar Practice App

A project I'm planning based on how I personally practice and improve on guitar.

The problem

Learning guitar has felt inconsistent. Tabs don’t always give the full picture, tutorials are slow, and theory is often hard to understand. It’s difficult to turn a song you like into something clear and useful to practice.

The idea

I want to build an app where a user can input a song and instantly get clean chords or tabs. The focus is not just generating notes, but making something that is actually playable and useful for improving. I also want to include a simple way to learn guitar theory quickly, without overwhelming the user.

Another issue I have come across is trying to memorize the fretboard's notes. I want to come up with a super interactive and fun way for users to be able to learn their guitar so they can play to the best of their ability.

Why I'm building it

I’ve found that the fastest way to improve is by playing songs I actually enjoy, but the current tools make that harder than it should be. This project is about making practice faster, clearer, and more engaging. I haven’t found a tool that fully solves this problem, and I want to make practicing guitar more direct and enjoyable instead of frustrating.
